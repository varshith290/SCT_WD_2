# SCT_WD_2

## Stopwatch Web Application

This project is created for SkillCraft Technology Web Development Task 2.

## Task Description

Create an interactive and user-friendly stopwatch web application. The stopwatch includes functions for starting, pausing, resetting, and tracking lap times.

## Features

- Start stopwatch
- Pause stopwatch
- Reset stopwatch
- Record lap times
- Display lap count
- Responsive user interface

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

Open `index.html` in a browser or run the project using the Live Server extension in VS Code.
